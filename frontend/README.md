# Handala Travels UI (React + TypeScript)

Multi-page travel website built with:

- React + TypeScript
- Material UI
- axios
- Jotai
- react-responsive-carousel
- react-router-dom

## Pages

- Home
- Ticket (fare table)
- Vehicles (bookable cards)
- Gallery
- Contact
- Account

## Run

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```
